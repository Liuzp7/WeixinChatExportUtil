    使用说明
    
        
      
          
    • 微信 4.1.10+ 需先构建内置 Hook 模块（npm run build:hook），并以管理员身份运行；也可导入 all_keys.json。
      • 
          
    • 若解密失败，请勾选「登录时捕获密钥」：工具会关闭微信，在您重新登录时自动提取密钥（微信 4.1.10+ 必需）。
      • 
          
    • 请先启动并登录微信 PC 版（Weixin.exe 需在运行中）。
      • 
          
    • 选择 xwechat_files 目录即可，工具会自动扫描并列出其中的微信账号。
      • 
          
    • 若目录下只有一个账号，会自动选中；多个账号时请在下拉框中选择。
      • 
          
    • 导出结果包含 conversations.json 索引和 chats/*.json 各会话文件。
      • 
          
    • 打包后的 exe 可直接发给他人使用，无需安装任何运行环境。
